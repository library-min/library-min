## About Me

안녕하세요.  
저는 **팀의 문제를 기술로 해결하는 풀스택 개발자 서재민**입니다.

단순히 기능을 구현하는 데 그치지 않고,  
사용자와 팀이 실제로 겪는 불편을 이해한 뒤  
더 나은 방향으로 개선하는 것을 중요하게 생각합니다.

K-Digital Training을 통해 웹·앱 개발 전반을 체계적으로 학습했고,  
**Neighbus 프로젝트**에서는 관리자 시스템 설계부터 모바일 앱 개발까지 직접 맡으며  
프론트엔드와 백엔드를 아우르는 실무형 경험을 쌓았습니다.
